### Setup
#### `.env` requirements
* `PORT` - Port 3000
* `MONGODB API` - https://mongo-api-server.herokuapp.com/api/v1/todo

#### Running the app
* `npm start`
* Endpoint: `/api/v1/todo/`
  * Returns a JSON object.
* Required data:
  * text - text
  * category - text
  * assignee - text
  * difficulty - number
  * complete - boolean
